# Ngobs

A real-time, AI-powered chat application built with **Java** and **Android Studio**. GeminiChat integrates **Google Gemini AI** to provide intelligent, context-aware responses in a seamless chat interface. The app features **secure phone number authentication** and **persistent storage** via Firebase.
